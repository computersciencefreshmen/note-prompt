# Note Prompt

一个现代化的提示词管理和分享平台，基于 Next.js 15 构建。

## 🚀 功能特性

- **公共提示词浏览** - 浏览和搜索社区分享的提示词
- **个人提示词管理** - 完整的 CRUD 操作管理您的提示词
- **一键导入** - 从公共库一键导入提示词到个人收藏
- **AI 优化** - 智能优化提示词效果（预留接口）
- **分类管理** - 按文件夹和标签组织提示词
- **响应式设计** - 支持桌面和移动设备

## 🛠️ 技术栈

- **框架**: Next.js 15 (App Router)
- **UI 组件**: shadcn/ui + Radix UI
- **样式**: Tailwind CSS
- **语言**: TypeScript
- **包管理**: Bun
- **部署**: Vercel

## 📦 安装和运行

```bash
# 安装依赖
bun install

# 启动开发服务器
bun run dev

# 构建生产版本
bun run build

# 启动生产服务器
bun start
```

## 🎯 页面结构

- `/` - 首页重定向到公共提示词页面
- `/public` - 公共提示词浏览页面
- `/prompts` - 个人提示词管理页面
- `/prompts/new` - 创建新提示词
- `/prompts/edit/[id]` - 编辑提示词

## 🔧 API 接口

项目预留了完整的 API 接口，支持：
- 获取公共提示词列表
- 个人提示词 CRUD 操作
- 提示词导入功能
- AI 优化接口（预留）

本项目为前端展示版本，API 接口为占位符实现。在生产环境中需要连接真实的后端服务。
